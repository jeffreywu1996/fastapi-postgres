# FastAPI with Postgres and Pytest
FastAPI with Postgres and Pytest

Source: https://medium.com/@arturocuicas/fastapi-with-postgresql-part-2-4a5eb435576a

An example of fastapi using postgres with great coding structure

```Shell
# Set up .env file
cp .env.example .env

# Start app
make run

# Teardown
make down

# Tests
make tests
```
